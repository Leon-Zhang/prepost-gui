iRIC 3.0 SDK README

○ コンパイル方法

ソースコードと、ライブラリ、ヘッダファイルを同じフォルダに置き、そのフォルダで
コンパイラにパスを通したコマンドプロンプトを開いて以下のコマンドを実行することで、
実行ファイルが生成されます。

ifort (ソルバのソースファイル) cgnsdll_x64_ifort.lib iriclib_x64_ifort.lib /MD

○ 実行方法

ビルドされた実行ファイルは、 iRIC とともに配布される DLL に依存しています。
iRIC がインストールされたフォルダ (例: C:\Users\User\iRIC\guis\prepost)
にパスを通した上で、起動して下さい。
