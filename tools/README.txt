README.txt

The python scripts in this folder are the files to be used in iRIC development processes.

== Commands ==

copydlls.py
  copy DLL files to libdlls\debug, libdlls\release.

qmake.py
  execute qmake in all sub directories of src.

qmakevc.py
  execute qmake -tp vc in all sub directories of src.

== Libraries ==

iricdevtool.py
  iric development tool library.
